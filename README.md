EcoSpot je nativna Android aplikacija razvijena primarno u Kotlinu kako bi se osigurao čist, moderan i pouzdan objektno-orijentisan kod. Koristio sam Firebase (autentifikaciju i bazu podataka) za brzu implementaciju sigurnog logovanja i sinhronizaciju podataka u realnom vremenu, čime sam izbegao pravljenje pozadinskog sistema (backenda) od nule. Takođe, Firebase Cloud Functions su korišćene za efikasno izvršavanje logike na serverskoj strani, dok je korisnički interfejs kreiran koristeći standardni Android SDK / XML.
